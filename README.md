# 📅 Lesson Calendar

Modern, şık ve kullanıcı dostu bir sınav takip uygulaması. Sınavlarınızı planlayın, geri sayımları takip edin ve hiçbir sınavı kaçırmayın!

![Premium Design](https://img.shields.io/badge/Design-Premium-blueviolet)
![Vanilla JS](https://img.shields.io/badge/JS-Vanilla-yellow)
![Responsive](https://img.shields.io/badge/Responsive-Yes-brightgreen)

## ✨ Özellikler

- **Sınav Yönetimi:** Sınav adı, tarih ve saat bilgilerini kolayca ekleyin.
- **Canlı Geri Sayım:** Her sınav için gün, saat ve dakika bazında gerçek zamanlı geri sayım.
- **Akıllı Durum Bildirimleri:** 
  - 🟠 **Kritik:** Sınava 2 günden az kaldıysa renk değişimi ile uyarı.
  - 🟢 **Devam Ediyor:** Sınav başladığında "Sınavınız Başladı!" bildirimi.
  - ⚪ **Geçti:** Tamamlanan sınavlar için pasif görünüm.
- **Veri Saklama:** Tarayıcı tabanlı `LocalStorage` kullanımı sayesinde verileriniz asla kaybolmaz.
- **Modern Arayüz:** Glassmorphism ve canlı gradyanlar içeren karanlık tema tasarımı.
- **Responsive:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu.

## 🚀 Kurulum

Projeyi yerelinizde çalıştırmak için sadece dosyaları bir tarayıcıda açmanız yeterlidir. Herhangi bir kurulum veya bağımlılık gerektirmez.

```bash
git clone https://github.com/mertarslna/LessonCalendar.git
cd LessonCalendar
open index.html
```

## 🛠️ Teknolojiler

- **HTML5:** Semantik yapı ve SEO dostu içerik.
- **CSS3:** Flexbox, Grid, Glassmorphism ve Modern Animasyonlar.
- **JavaScript (ES6+):** Saf (Vanilla) JS ile modüler uygulama mantığı.
